## Upgrading Bertus Erlank

Follow these steps to upgrade to the latest version. Always back up your projects first.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command in your terminal.
    
````bash
npx bertus-erlank --version
````
  </Step>
  <Step title="Upgrade Package" icon="download">
    Use your preferred package manager.
    
    <CodeGroup tabs="npm,yarn,pnpm">
````bash
npm install bertus-erlank@latest
````
````bash
yarn add bertus-erlank@latest
````
````bash
pnpm add bertus-erlank@latest
````
    </CodeGroup>
  </Step>
  <Step title="Verify Installation" icon="check-circle">
    Test the upgrade with a sample project.
    
````bash
npx bertus-erlank init my-project
cd my-project
npx bertus-erlank dev
````
  </Step>
</Steps>
